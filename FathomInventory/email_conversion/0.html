Meeting Summary ✨

Meeting Purpose

Explore collaboration opportunities between Restor.eco and the Big Map to Save the Future project

Key Takeaways

  • Restor.eco and Big Map have complementary strengths in GIS/data and storytelling/inspiration respectively

  • Both parties are interested in data sharing and potential API integration to cross-populate projects

  • Gwyn from Restor.eco will join an upcoming ERA town hall meeting to discuss collaboration further

  • Exploring options for Restor.eco to support Big Map financially in a mutually beneficial way

Topics

Restor.eco Overview

  • Platform for mapping restoration/conservation projects globally

  • Provides GIS data layers and tools for users to add project polygons

  • Focuses on data accuracy and scientific metrics

  • Has a team of "local leads" to onboard projects worldwide

  • Looking to expand to include plant nurseries, seed banks, suppliers

Big Map to Save the Future Overview

  • Interactive map showcasing restoration projects and success stories

  • Focuses on inspiration, aspiration and lowering barriers to entry

  • Aggregates data from various sources (news articles, partner orgs)

  • Working on automating data collection/entry process

  • Part of broader Eco Restoration Alliance (ERA) initiative

Data Sharing Opportunities

  • Big Map interested in accessing Restor.eco's public project data

  • Discussed potential for API or database export of public entries

  • Restor.eco open to exploring options, needs to build case internally

  • Big Map may attempt to scrape data in interim if needed

Mutual Benefits of Collaboration

  • Cross-promotion to grow user bases on both platforms

  • Restor.eco could provide GIS expertise to Big Map projects

  • Big Map could help surface compelling stories from Restor.eco data

  • Potential to jointly apply for funding opportunities

X Prize Competition

  • Big Map interested in leveraging Restor.eco data for carbon drawdown metrics

  • Could strengthen application for $25M prize in 2024

  • Restor.eco open to exploring this opportunity further

Next Steps

  • Explore options for Restor.eco financial support of Big Map project

  • Jon to attempt scraping public Restor.eco data as interim solution
